# Linux Basics Lab

## Objective
Explore Linux system information and understand server details.

## Commands Executed

### Host Information
hostnamectl

### Kernel Information
uname -a

### Operating System
cat /etc/os-release

### Memory Information
free -h

### Disk Information
df -h

### Block Devices
lsblk

### Network Information
ip addr

## Learning Outcome

- Learned how to identify Linux distribution.
- Learned how to check memory and storage.
- Learned how to inspect network configuration.
- Learned basic Linux navigation commands.
