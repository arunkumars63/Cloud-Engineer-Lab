# Basic Linux Commands

## Objective

Learn basic Linux commands used by system administrators.

## Commands Practiced

### Check Current Directory

```bash
pwd
```

### List Files

```bash
ls
```

### Create Directory

```bash
mkdir testfolder
```

### Change Directory

```bash
cd testfolder
```

### Remove Directory

```bash
cd ..
rmdir testfolder
```

## Learning Outcome

Learned how to navigate and manage files and folders in Linux.
